Elite High-Fidelity Interactive Flex Accordion Menu for WordPress.

== Description ==
A professional, high-performance accordion menu system designed for modern websites. Purified engine, professional assets, and industry-standard naming.

== Installation ==
1. Upload the plugin folder to the /wp-content/plugins/ directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
